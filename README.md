# Tata Consumer Products Financial Dashboard (Python)

## Project Overview
Built an executive financial dashboard analysing multi-year revenue growth, EBITDA margins, CAGR, and key financial ratios.

## Key Metrics
- Revenue CAGR
- EBITDA Margin %
- Net Profit Margin
- ROE, ROA
- Debt-to-Equity Ratio

## Tools Used
- Python
- Pandas
- Matplotlib
- Financial Modelling Techniques

## Business Insights
Converted financial statements into structured executive-level insights to evaluate growth sustainability and capital efficiency.
